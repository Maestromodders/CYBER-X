# Security Policy — CYBER-XMD

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in CYBER-XMD, please open an issue or contact the repository owner directly. We take security seriously and will respond promptly to any reports.

- Do **not** publicly disclose vulnerabilities before they are patched.
- Provide as much detail as possible (steps to reproduce, impact, etc.).
- You can expect an acknowledgment within 48 hours.
