## ⚙️ Configuration (app.json / Environment Variables)

| Variable | Description | Default |
|---|---|---|
| `SESSION_ID` | Your WhatsApp session ID (required) | — |
| `BOT_NAME` | Name of the bot | `CYBER-XMD` |
| `PREFIX` | Command prefix | `.` |
| `MODE` | Bot mode: public/private/inbox/groups | `public` |
| `OWNER_NUMBER` | Owner WhatsApp number (no +) | — |
| `OWNER_NAME` | Owner display name | `CYBER-XMD Owner` |
| `AUTO_REPLY` | Auto-reply to messages | `false` |
| `READ_MESSAGE` | Auto read messages | `false` |
| `AUTO_REACT` | Auto react to messages | `false` |
| `ALWAYS_ONLINE` | Always show online | `false` |
| `AUTO_TYPING` | Show typing indicator | `false` |
| `ANTI_LINK` | Anti-link protection | `true` |
| `ANTI_DELETE` | Anti-delete messages | `true` |
| `ANTI_CALL` | Block incoming calls | `false` |
| `WELCOME` | Send welcome messages | `false` |
| `GOODBYE` | Send goodbye messages | `false` |
| `AUTO_STATUS_SEEN` | Auto-view statuses | `true` |
| `AUTO_DOWNLOADER` | Auto download media | `false` |
| `DESCRIPTION` | Bot footer description | `⚡ Powered by CYBER-XMD` |
